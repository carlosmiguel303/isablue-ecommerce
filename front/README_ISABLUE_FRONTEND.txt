ISA BLUE - FRONTEND ANGULAR

Pasos para ejecutar:

1. Abrir terminal en esta carpeta:
   ecommerce-full-stack-front-main

2. Instalar dependencias solo la primera vez:
   npm install

3. Ejecutar Angular:
   ng serve

4. Abrir:
   http://localhost:4200

Cambios incluidos:
- Banner principal tipo Coolbox con carrusel Bootstrap.
- Franja oscura de categorías.
- Cards debajo del banner con ofertas y secciones comerciales.
- Branding IsaBlue Juguetes.
- Bootstrap JS agregado en angular.json para que funcione el carousel.

Backend esperado:
http://localhost:8085
